NanotecSMCPD Instrument Driver Readme

To rate this instrument driver, go to the driver download page on the Instrument Driver Network.

1. Overview

Instrument Driver Technology: LabVIEW Plug and Play (project-style)
Manufacturer: Nanotec
Supported Language(s): LabVIEW
Supported Model(s): SMCI32, SMCI47, PD4-I and PD6-I
Model(s) Tested: SMCI32
Interface(s): Serial
Firmware Revision(s) Tested: 04-12-2008

Certified: No
NI Supported: No
Source Code Available: Yes
License: GNU Public License

Driver Revision: see version number of library
Original Release Date: see revision history below
Current Revision Date: see revision history below

2. Required Software

Some software components need to be installed before using this instrument driver. The minimum versions of these components are listed below, and can be downloaded from the Download Site.

NI-VISA 4.1 or later, NI-Serial 3.2.0 or later

Refer to the LabVIEW Help for more information about software requirements. You access the LabVIEW Help by selecting Help»Search the LabVIEW Help.

3. Known Issues

To report issues or provide feedback about this instrument driver, please send an email to d.beck@gsi.de. This driver just implements a sub-set of the functionality supported by the device.
